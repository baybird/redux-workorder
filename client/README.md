
#Create and run app
```
$ npm install -g create-react-app
$ create-react-app my-app
$ cd my-app/
$ npm start
```

#Create a production build
```
$ npm run build
```

#run production
```
nodemon production.js
```

#run development
```
npm start
```

#Change port for development
```
# Windows - Powershell
$ SET PORT=80

#Linux
$ export PORT=3005 
```
